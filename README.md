# Study Time Tracker

This Python program helps you estimate your weekly study time based on your daily hours. It takes user input, performs calculations, and provides clear results. It also includes simple error handling so the program doesn't crash if the user enters invalid input.

## How to Run
python app.py  
# or  
python3 app.py

## Example Output
Enter hours studied today: 2  
You are on track to study 14 hours this week.
